# Bug Report: BUG-007

## Summary

Password reset link expires immediately after it is generated.

## Module

Forgot Password

## Severity

High

## Priority

High

## Steps

1. Click Forgot Password.
2. Request reset email.
3. Open the email.
4. Click the reset link.

## Expected Result

Reset page opens.

## Actual Result

Expired Link message appears.

## Status

Open
