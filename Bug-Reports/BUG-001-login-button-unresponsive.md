# Bug Report: BUG-001

## Summary

The **Login** button does not respond when valid credentials are entered.

---

## Bug ID

BUG-001

## Module

Login

## Environment

- Application: E-Commerce Web Application
- Environment: QA
- Browser: Google Chrome (Latest)
- Operating System: Windows 11

---

## Severity

**High**

## Priority

**High**

---

## Preconditions

- User account exists.
- User is on the Login page.

---

## Steps to Reproduce

1. Open the Login page.
2. Enter a valid email address.
3. Enter the correct password.
4. Click the **Login** button.

---

## Expected Result

The user should be logged in successfully and redirected to the Home page.

---

## Actual Result

Nothing happens after clicking the Login button. The user remains on the Login page.

---

## Reproducibility

Always (100%)

---

## Status

Open

---

## Notes

This issue prevents users from accessing their accounts and should be fixed before release.
