This repository contains manual testing artifacts including:

- Test Scenarios
- Test Cases

- Test Data
- Requirements Traceability Matrix (RTM)

# 📋 Testing Artifacts

| Document | Description |
|----------|-------------|
| Test Plan | Defines the testing strategy, scope, objectives, and approach. |
| Test Scenarios | High-level scenarios covering key application functionality. |
| Test Cases | Detailed manual test cases with expected results. |
| Bug Reports | Defects documented with severity, priority, and reproduction steps. |
| Test Data | Positive, negative, and boundary value test data used during testing. |
| RTM | Maps business requirements to test scenarios, test cases, and defects. |
| Test Summary Report | Summarizes the testing cycle, execution, and outcomes. |
| Defect Summary Report | Provides an overview of identified defects and their status. |
